关于最近有关于ss可以被探测以及相关论文的一点看法

发表于 2017-10-12

起因

最近出现一些消息说ss可以被探测，且正确率可以达到75%之多，这项技术是由阿里云和360联合开发的，还写了论文，作为一个爱国人士，我迫不及待的看了一下这篇论文，看看是不是我们国家的科技水平发展的这么迅速，是不是我们都落后了。

经过

没有。是的，没有经过，经过就是我看了一下那篇论文。

正文

最近有两篇号称是检测 Tor 和 Shadowsocks 的论文。四个字，纯属放屁，当毕业设计都不一定及格。大家不用在上面浪费时间了。先不说论文作者认为「Shadowsocks 是用 SSH 连接的」，那论文连单词都拼不对（据说这已经是重写过的）。建议作者回到高中重新学习英文，或者给我 50 块钱让我代译英文版也行 [doge]

说道滥用问题，作者说：

To effectively prevent these things happened

「prevent things happened」，要么是作者重新发明了英文语法，要么是作者发明了时间机器。

提到使用代理访问新闻的现象，作者说：

the news even contain sensitive information in politics, economics, democratic, financial, technology and so forth.

先不说「contain」这个动词的问题，「politics」和「economics」还说得过去，都是名词。但「democratic」和「financial」是什么鬼，怎么突然变形容词了，这是什么语言？还「so forth」，什么鬼？

还说：

more and more people has learnt

看来不仅应该回到高中，还应该先回初中。

提到准确率的问题，作者说：

test set and feature set to reduce the false alarm rate and false rate when detecting

「false alarm rate and false rate」是什么鬼？这 false rate 好像说了两次？这体现了论文作者的最高水平——根本不知道 false-positive 和 false-negative 两个单词，统计学估计是不及格的。

说道检测的重要性，作者说：

do the following decrypted work and content analysis.
「随后进行解密工作」变成了「do decrypted work」，我还「encrypted work」呢……听说论文作者是研究网络安全的，听没听说过一种非常尖端的科技，叫做这个——「cryptoanalysis」？

唯一没什么问题的，是这半句话：

and to immediately detect and arrest these criminals

好大的口气！「immediately detect and arrest criminals!」「immediately! 」看来说的是真心话，语法一点问题没有。

以及：

Shadowsocks make proxy by using established channel, which means that requesting the real servers by SSH server. Lastly, Servers through SSH server send back the responding data by using established channel. Since SSH itself is based on RSA……

说 Shadowsocks 是通过 SSH 连接的……作者可能分不清「通过 SSH 进行网络连接」以及「装了一个 SSH 能管理系统」的服务器有什么区别。或者说……难道作者这篇论文是故意放水的，发扬了枪口抬高一光年的精神，真是令人肃然起敬。

这两篇文本的 DOI 编号分别为：

10.1109/IHMSC.2017.132
10.1109/IHMSC.2017.133

大家可以去 sci-hub.cc 输入 DOI 直接下载盗版 PDF，打印出来，挂在你的学校、办公室或者实验室，提醒自己和同行：如果不加倍努力，将来你的水平就会像这篇论文一样。
